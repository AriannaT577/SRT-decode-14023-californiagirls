# FTC Team 14023 (SRT Phoenix Resurrection) 

Our Autonomous code for the DECODE 2025-2026 season. This version is the final version of the autonomous code, for when we competed in the Worlds Western Edge Premier Event in California. 

Our Autonomous codes can be found in TeamCode/src/main/java/org/firstinspires/ftc/teamcode. Here you can see our autonomous files (In Auto) and our tuning files (MecanumDrive.java and ThreeDeadWheelLocalizer.java) for Maddison V, the robot.
